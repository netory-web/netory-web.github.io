주요 기능

빠른 연락처 검색

이름, 그룹, 키워드 기반으로 원하는 연락처를 빠르게 탐색할 수 있습니다.

그룹 관리

가족, 친구, 업무, 거래처 등 다양한 그룹으로 체계적인 관리가 가능합니다.

간편한 UI

누구나 쉽게 사용할 수 있도록 직관적인 인터페이스를 제공합니다.

Netory 소개

Netory는 복잡한 연락처 관리의 불편함을 해결하기 위해 만들어졌습니다. 연락처를 더 효율적으로 정리하고 빠르게 찾을 수 있도록 설계되었으며, 사용자 중심의 간단하고 편리한 경험을 제공합니다.
